# 项目简介

## 核心功能
- 会员登录模块：支持账号密码和验证码登录注册
- 个人中心：展示并允许修改个人基本信息，如账号密码，电话地址，车牌等
- 商品管理：提供沙箱支付，收藏，购买商品赠送积分，积分兑换商品，评论功能
- 营销活动：展示多种活动提示，通知会员商品优惠和营销力度
- 推荐系统：基于用户消费习惯和偏好，使用协同过滤算法推荐商品
- 发票处理：提供发票开具服务
- 购物车：加入商品至购物车，生成订单，查看订单状态和记录
- 管理员模块：
  - 查看和管理会员信息及账号
  - 管理商品及积分商品，查看评论并处理不当评论
  - 查看订单状态，统计订单完成情况
  - 统计商品信息，生成报表
  - 发布优惠活动
  - 统计会员信息，导出会员数据
  - 编辑、禁用/启用会员账号
  - 批量导入与导出会员数据
  - 系统自动计算积分，提供积分收支明细查询
  - 设置积分有效期，自动清理过期积分
  - 展示即将过期的积分并提醒用户

## 适用场景
- 加油站会员管理系统
- 适用于需要会员管理、商品销售、营销活动推广、积分系统等功能的场景

## 技术特点
- 基于 Java 开发
- 前后端分离，易于维护和扩展
- 使用沙箱支付，保障交易安全
- 提供丰富的营销活动设置，提高用户活跃度
- 强大的数据分析能力，帮助管理员优化运营策略
- 系统自动处理积分，减轻管理员工作负担
